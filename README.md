# TNM093-VIS-APPLICATION  
**Visualiseringsprojekt inom kursen TNM093**

## Om projektet
Detta projekt är en webbaserad visualiseringsapplikation utvecklad inom kursen **TNM093. Visualiseringstekniker**. Syftet med projektet är att utforska hur data och beräknade värden kan representeras visuellt på ett tydligt och interaktivt sätt i en webbläsare.

Applikationen använder JavaScript för att bearbeta data och generera grafiska representationer som gör det möjligt att analysera samband, förändringar och mönster visuellt. Fokus ligger på grundläggande visualiseringsprinciper såsom tydlighet, struktur och användarinteraktion.

## Hur man använder
1. Klona eller ladda ner repositoryt  
2. Öppna `index.html` i en modern webbläsare  
3. Visualiseringen visas direkt och kan interageras med enligt labbens instruktioner  

Ingen installation krävs. All funktionalitet körs lokalt i webbläsaren.
